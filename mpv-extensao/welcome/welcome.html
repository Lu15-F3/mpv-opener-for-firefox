Quase pronto! Configuração Necessária 🐧

O MPV Opener for Firefox foi instalado no seu navegador, mas ainda não consegue se comunicar com o seu sistema Fedora. Siga os dois passos rápidos abaixo:

1. Instalar as dependências do Fedora

Abra o seu terminal e rode o comando para instalar o player, o extrator e o plugin de mídia:

sudo dnf install mpv yt-dlp mpv-mpris

2. Ativar a ponte de comunicação (Native Host)

Agora execute o instalador oficial no seu terminal para configurar os manifestos de integração locais:

curl -sSL https://raw.githubusercontent.com/Lu15-F3/mpv-opener-for-firefox/main/install.sh | bash

Após rodar os comandos, a extensão funcionará instantaneamente sem precisar reiniciar o Firefox.
