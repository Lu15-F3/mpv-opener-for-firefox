Almost Ready! Setup Required 🐧

The MPV Opener for Firefox has been added to your browser, but it cannot communicate with your Fedora system yet. Follow these two quick steps:

1. Install Fedora Dependencies

Open your terminal and run this command to install the player, extractor engine, and media plugin:

sudo dnf install mpv yt-dlp mpv-mpris

2. Enable the Communication Bridge (Native Host)

Now execute the official installer script to register the local integration manifests:

curl -sSL https://raw.githubusercontent.com/Lu15-F3/mpv-opener-for-firefox/main/install.sh | bash

After running the commands, the extension will work instantly. No need to restart Firefox.

Quase Pronto! Configuração Necessária 🐧

O MPV Opener for Firefox foi adicionado ao seu navegador, mas ainda não consegue se comunicar com o seu sistema Fedora. Siga estes dois passos rápidos:

1. Instalar as Dependências do Fedora

Abra o seu terminal e rode o comando para instalar o player, o motor extrator e o plugin de mídia:

sudo dnf install mpv yt-dlp mpv-mpris

2. Ativar a Ponte de Comunicação (Native Host)

Agora execute o script instalador oficial para registrar os manifestos de integração locais:

curl -sSL https://raw.githubusercontent.com/Lu15-F3/mpv-opener-for-firefox/main/install.sh | bash
